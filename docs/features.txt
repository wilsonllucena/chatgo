Criar um sistema para auxiliar professores a criarem execicios para suas aulas.

Regras de negócio:
    [x] A entrada do sistema será via chat, onde o professor irá enviar mensagens para o sistema.
    [x] O sistema irá responder as mensagens do professor com os exercícios criados.
    [ ] O sistema irá salvar os exercícios criados em um banco de dados.
    [x]O sistema irá exibir os exercícios criados em uma interface web.
    [x] O sistema irá exibir as respostas dos exercícios em uma interface web.
    [ ] O sistema terá funcionalidade para gerar o arquivo de exercícios em formato PDF e Word.
